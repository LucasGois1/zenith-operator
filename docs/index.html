Made Simple

Orchestrate builds, deployments, and event-driven invocations with a single Custom Resource. Abstract the complexity of Tekton, Knative, and Dapr.

Get Started Learn More
function.yaml 
apiVersion: functions.zenith.com/v1alpha1
kind: Function
metadata:
  name: payment-processor
spec:
  gitRepo: https://github.com/org/payments
  build:
    image: registry.io/payments:v1
  deploy:
    dapr:
      enabled: true
      appID: payment-service
  eventing:
    broker: default
    filters:
      type: order.created

Why Zenith?

Automated Builds

Tekton Pipelines integrated with Cloud Native Buildpacks. Source to image without a Dockerfile.

Serverless Deployment

Knative Serving under the hood. Auto-scaling, scale-to-zero, and traffic splitting out of the box.

Event Driven

Native Knative Eventing support. Subscribe to brokers and filter events with simple YAML config.

Dapr Integration

First-class support for Dapr sidecars. Enable service mesh capabilities with a single boolean flag.

GitOps Ready

Everything is declarative. Manage your serverless functions using standard Kubernetes manifests.

Secure by Design

Secret-based authentication for Git and Registries. Non-root containers by default.

Installation

Deploy Zenith Operator and all dependencies to your Kubernetes cluster using Helm.

# Add the Helm repository
helm repo add zenith-operator https://lucasgois1.github.io/zenith-operator
helm repo update

# Install the operator and all dependencies
helm install zenith-operator zenith-operator/zenith-operator

This automatically installs Tekton, Knative, Kong, and all required dependencies.

Quick Start

Deploy your first function using a public Git repository.

apiVersion: functions.zenith.com/v1alpha1
kind: Function
metadata:
  name: hello-world
spec:
  gitRepo: https://github.com/LucasGois1/zenith-test-functions
  build:
    image: docker.io/your-username/hello-world
  deploy:
    dapr:
      enabled: true
      appID: hello-world
      appPort: 8080

API Reference (v1alpha1)

FunctionSpec

Field Type Description
gitRepo string (required) URL of the Git repository containing source code.
gitRevision string Git branch, tag, or commit hash (default: main).
gitAuthSecretName string Secret for private Git repository authentication.

BuildSpec

Field Type Description
image string (required) Target container image name.
registrySecretName string Secret for container registry authentication.

DeploySpec

Field Type Description
dapr.enabled boolean Inject Dapr sidecar.
dapr.appID string Dapr application ID.
dapr.appPort int Port the function application listens on.
